# MCloud - Chat

## Installation

### Step 1: Install [MCloud](https://mcloud.io)

### Step 2: Init app

``` bash
mcloud init chat [path_to_mcloud_chat]
```

### Step 3: Start app

``` bash
mcloud start chat
```

### Step 4: Show status

``` bash
mcloud status chat
```
